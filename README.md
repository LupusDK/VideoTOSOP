# Video to SOP 分析助手

本專案利用 Google Gemini API 的原生多模態能力，直接分析影片內容並自動生成標準作業程序（SOP）文案。

## 🛠️ 使用前準備

1. **準備 Google Gemini API KEY**
   - 本專案優先支援 Gemini API，主因在於其具備原生影像辨識能力，可直接處理影片檔案，無需額外進行影音分離或繁瑣的預處理。

2. **金鑰偵測與模型相容性**
   - 輸入 API KEY 後，請點擊 **「偵測」** 按鈕。
   - 系統將確認金鑰效力，請確保帳戶擁有 **Gemini 1.5 Flash**（含）以上等級之模型權限。

## 💡 操作指引

### SOP 分析指令 (Prompt) 設定
- **預設指令**：可直接使用內建預設 Prompt 進行分析。
- **角色定義 (Role Prompting)**：若希望 AI 具備特定產業知識，建議在 Prompt 中增加「角色定義」。
  - *範例*：「你是個精通 SAP 產品及 ERP 流程的專家，請根據影片內容分析並撰寫 SOP。」

### 編輯與匯出
右側產出的 SOP 內容均支援即時編輯，完成後可選擇以下方式處理：
- 📄 **匯出 PDF**：產出標準格式文件。
- 📝 **複製 Markdown (.md)**：方便貼至 GitHub 或其他筆記軟體。
- ⚙️ **匯出 JSON**：適合串接其他 AI 代理或自動化工具進行二次加工。
